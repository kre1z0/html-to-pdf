12. Список документов для рассмотрения заявки на кредит

Основные

  • Бизнес-план
  • Заявка на кредит (по форме банка)
  • Гарантия Корпорации МСП
  • Краткие сведения об инициаторах бизнеса, включая паспортные данные
  • Обеспечение по кредиту

Правоустанавливающие

  • Свидетельство о праве собственности объекта недвижимости
  • Договор аренды площадей, используемых в бизнесе
  • Заключение Госпожарнадзора/МЧС, СЭЗ Роспотребнадзора

Учредительные

  • Копия свидетельства о постановке на налоговый учет (ИНН)
  • Копия свидетельства о присвоении ОГРН
  • Выписка из ЕГРЮЛ / реестра МСП ФНС
  • Копии лицензий, патентов, разрешений
  • Копии учредительных документов (для юрлиц)

Отчетные

  • Система налогообложения: ОСН, УСН, ЕНВД; ПО: нет / 1С / Абакус / …
  • Баланс (форма №2) за последний год и последний квартал
  • Копия свидетельства об уплате ЕНВД / налога по УСН за последний период
  • Книга учета доходов и расходов
